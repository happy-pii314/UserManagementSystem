# User Management System

A responsive User Management System built using HTML, CSS, JavaScript, PHP, and MySQL.

# User Management System

🚀 **Live Demo:** http://usermanagement.wuaze.com

💻 **GitHub Repository:** https://github.com/happy-pii314/UserManagementSystem

## Features

- Add User
- View Users
- Update User
- Delete User
- Search Users
- Responsive Design
- REST API
- MySQL Database

## Technologies Used

- HTML5
- CSS3
- JavaScript
- PHP
- MySQL
- XAMPP

## Project Structure

```
UserManagementSystem/
│
├── index.html
├── style.css
├── script.js
├── api.php
├── users.sql
└── README.md
```

## How to Run

1. Install XAMPP.
2. Start Apache and MySQL.
3. Import `users.sql` into phpMyAdmin.
4. Copy the project folder to `htdocs`.
5. Open:

```
http://localhost/UserManagementSystem/
```
